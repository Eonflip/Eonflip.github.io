Command Line Interface 101

If you're here reading this I hope your curious about what the command line interface is I certainly was before a few years ago.
I had used computers my entire life starting with an old computer running windows 95 my parents had lying around
or the wonderful imac G3's that lining my elementary schools computer lab. I'm dating myself but the point is I had never touched the command line interface until about 4 years ago.
Now I could never imagine going back.

What is the command line interface?

The command line interface is a program running on most every computer that you'll interact with. It allows you to interact at a deeper level with the computer.
We're all used to graphical user interfaces or GUIs to manage our day to day tasks, your wonderful desktop and web browser you're reading this from right now are examples of graphical user interfaces
The command line interface however can appear a bit more arcane such as this

This may look scary but let's break this down a bit, everything running on your computer is essentially a collection of text that gets converted into computer readable code or instructions to tell the computer what to do.
Instead of being limited by a GUI sometimes we can get a little bit deeper with the command line interface.
Whether your on a Windows, Mac, or Linux Machine you'll be able to access it.
